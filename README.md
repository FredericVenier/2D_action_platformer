# 2D Action Platformer
2D Action platformer prototype (work in progress) made using C++ and SDL2 library.

------------------------------------------------------------------------------

CONTROLS :

    AD/QD to move
    
    S to crouch
    
    SPACE to jump

    K to attack

------------------------------------------------------------------------------

The linux verion is updated more often.
